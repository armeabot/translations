# Armeabot Translation Project

This repository contains the localization and translation files for the Armeabot project.

## 🚀 How to Contribute

You can follow these steps to contribute to the translations:

1. Open the `.json` file for the language you want to translate.
2. Click the ✏️ (edit) icon in the top right corner.
3. Edit or add your translations.
4. Click the **Commit changes...** button to save your work.

## ⚠️ Important Rules

- **Do not modify** the keys on the left side of the `.json` file. Only translate the text values on the right side.
- Pay close attention to quotation marks (`" "`) and commas (`,`) so you don't break the JSON format.

Thank you for your contributions! ✨
